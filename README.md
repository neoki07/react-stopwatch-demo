# React Stopwatch Demo

Reactで作成されたストップウォッチです。

## アプリのURL

https://react-stopwatch-demo.vercel.app/

## 機能

* ストップウォッチのスタート / ストップ
* ストップウォッチの時刻リセット
* ダークモード / ライトモードの切り替え

## 開発環境

* JavaScript
* Yarn
* React
* React Testing Library
* ESLint
* Prettier
* Tailwind CSS
* CircleCI
* Visual Studio Code
